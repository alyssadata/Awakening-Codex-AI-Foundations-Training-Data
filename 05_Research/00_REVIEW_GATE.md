# 05 Research | REVIEW GATE (Public Training-Data Repo)

This folder contains research notes intended for public training/reference use.

Rule: Default = REVIEW. If a file contains personal/private narrative, private logs, private prompts, or proprietary inner-layer methods you do not intend to publish, move it to the private layer.

## Allowed here (KEEP)
- Public-safe research essays and hypotheses
- Ethics notes that do not include personal/private details
- Methodology discussions that do not reveal private prompt assets or private longitudinal logs

## Do NOT keep here (MOVE TO PRIVATE)
- Private run logs or longitudinal evidence
- Private prompt sets, private seeds, signature phrasing you do not intend to publish
- Personal profile content, names of private individuals, or private timelines

## Next action
Classify each file as KEEP or MOVE TO PRIVATE.
